# paella-plugin-quizzes

paella-plugin-quizzes plugin for [Paella player](https://github.com/polimediaupv/paella).



## Build

    $ npm run build

## Demo

    $ npm run start

Go to `http://localhost:8080/player/?id=belmar-multiresolution-remote`
